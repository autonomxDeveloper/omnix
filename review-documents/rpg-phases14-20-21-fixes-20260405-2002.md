## Changes Made

### Phase 21: Performance Fixes

| Change | Description |
|--------|-------------|
| **Diagnostic-only module** | Added IMPORTANT note clarifying this module is strictly diagnostic; wall-clock timing must not be used as authoritative simulation inputs. |
| **VALID_METRIC_UNITS** | Added constant set `{"ms", "count", "bytes"}` for allowed metric units. |
| **_normalize_unit()** | New helper function that validates/normalizes metric unit strings. |
| **PerformanceMetric.from_dict()** | Clamps negative values to 0.0, invalid ticks to 0, normalizes units. |
| **PerformanceMetric.to_dict()** | Reformatted for readability. |
| **PerformanceState.record()** | Clamps negative values and ticks on input, normalizes units. |
| **PerformanceState.record_diagnostic_only()** | New explicit alias method for diagnostic-only recording intent. |
| **BenchmarkHarness.benchmark()** | Added `"diagnostic_only": True` flag to all benchmark payloads. |
| **PerformanceDeterminismValidator.validate_bounds()** | Extended to validate negative values, negative ticks, and invalid units. |
| **PerformanceDeterminismValidator.validate_diagnostic_only_payload()** | New validator for benchmark diagnostic_only flag. |
| **PerformanceDeterminismValidator.normalize_state()** | Now sorts metrics deterministically (tick, name, unit, value), clamps tick and budget_ms to non-negative. |

### Phase 20: Save/Packaging Fixes

| Change | Description |
|--------|-------------|
| **SaveSnapshot redesign** | Changed from `header`/`game_state`/`subsystem_states` to `save_id`/`version`/`tick`/`simulation_state`/`metadata`/`integrity_hash`. |
| **Stable JSON serialization** | Added `_stable_json_data()`, `_stable_json_dumps()`, `_stable_hash()` for deterministic ordering and hashing. |
| **_normalize_payload()** | Normalizes all state payloads through stable JSON conversion. |
| **_is_json_like()** | Validator helper to check if data is JSON-serializable. |
| **SAVE_FORMAT_VERSION** | Changed from 8 to 1 (cleaner baseline for new schema). |
| **MAX_MIGRATION_HISTORY / MAX_VALIDATION_ERRORS** | New constants for bounding history and error lists. |
| **MigrationRecord / SavePackagingState** | New dataclasses for migration tracking and packaging state management. |
| **SaveSchema.validate_snapshot_dict()** | Extended validation for integrity_hash, JSON-like structure, bounded errors. |
| **SaveBuilder** | New class with `build_snapshot()` and `compute_integrity_hash()`. |
| **SaveSerializer** | Rewritten to use stable JSON serialization and normalization. |
| **SaveMigrator** | Simplified migration with history tracking and limit enforcement. |
| **ReplayConsistencyChecker** | Updated to use normalized snapshots for round-trip checks. |
| **SavePackagingValidator** | New unified validator with integrity hash verification and normalization. |

### Phase 14: Director Bias-Only Fixes

| Change | Description |
|--------|-------------|
| **VALID_BEAT_TYPES / VALID_BEAT_STATUS / VALID_ARC_STATUS** | New constant sets for allowed enum values. |
| **_normalize_arc_phase()** | Normalizes arc phase to valid ARC_PHASES, defaults to "setup". |
| **_normalize_beat_type()** | Normalizes beat type to valid BEAT_TYPES, defaults to "event". |
| **_normalize_beat_status()** | Normalizes beat status to valid BEAT_STATUS, defaults to "pending". |
| **_normalize_arc_status()** | Normalizes arc status to valid ARC_STATUS, defaults to "active". |
| **_sort_key_beat() / _sort_key_arc()** | Deterministic sort key functions for beats and arcs. |
| **StoryBeat.from_dict()** | Uses normalization helpers for beat_type and status. |
| **StoryArcState.from_dict()** | Uses normalization helpers for phase and status. |
| **SceneBiasEngine docstring** | Clarified advisory-only nature; never mutates simulation state. |
| **DirectorDialogueInfluence docstring** | Clarified advisory-only nature; hints not authoritative overrides. |
| **DirectorQuestInfluence docstring** | Clarified advisory-only nature; never mutates quest state directly. |
| **DirectorDeterminismValidator.validate_bounds()** | Extended to validate arc phases, arc status, beat types, beat statuses. |
| **DirectorDeterminismValidator.normalize_state()** | Sorts arcs by priority desc then arc_id, sorts beats, normalizes focus_entities, clamps and normalizes all enum fields. |
| **director/__init__.py** | Updated to re-export all new classes and handle deprecated Director import. |

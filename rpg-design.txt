Goal

Allow the player to intervene in an active scene with actions like:

step in
calm them down
take a side
ask what is happening
redirect the topic
threaten / provoke
leave / ignore

Those interventions should:

target the active scene by scene_id
affect scene tension and progression
optionally resolve, redirect, or escalate the scene
stay deterministic and bounded
never let the LLM become simulation truth
Design model
Scene intervention is a runtime/presentation action

It is not raw simulation truth by itself.

Flow:

player selects an intervention
backend validates the scene is active and nearby
backend applies deterministic scene-state mutation
backend emits new scene beats / outcomes
optional world consequence follows
LLM role

The LLM may help phrase the resulting narration/dialogue, but it should not decide:

whether the player successfully calmed the scene
whether the scene resolves
whether tension rises or drops

That should be rule-based.

New concepts
Intervention action types

Add a small bounded set:

observe_scene
step_into_scene
ask_about_scene
support_primary
support_secondary
calm_scene
provoke_scene
redirect_scene
leave_scene

These are enough for a strong first pass.

Intervention effects

Each action changes scene state deterministically.

Example:

calm_scene → tension -0.15
provoke_scene → tension +0.18
support_primary → tension +0.08 or -0.05 depending on scene kind
ask_about_scene → converts player pull into direct explanation beat
leave_scene → marks player disengaged, allowing escalation to continue
redirect_scene → changes topic to player_intervention

+    scene_id = _safe_str(data.get("scene_id")).strip()
+    action_type = _safe_str(data.get("action_type")).strip()

+        return JSONResponse({"ok": False, "error": "scene_id_required"}, status_code=400)
+    if not action_type:
+        return JSONResponse({"ok": False, "error": "action_type_required"}, status_code=400)
+

+    if session is None:
+        return JSONResponse({"ok": False, "error": "session_not_found"}, status_code=404)
+
+    runtime_state = _safe_dict(session.get("runtime_state"))
+    simulation_state = _safe_dict(session.get("simulation_state"))
+
+    check = validate_scene_intervention(runtime_state, simulation_state, scene_id, action_type)
+    if not check.get("ok"):
+        return JSONResponse(check, status_code=400)
+
+    result = apply_scene_intervention(
+        runtime_state,
+        simulation_state,
+        scene_id=scene_id,
+        action_type=action_type,
+    )
+
+    session["runtime_state"] = _safe_dict(result.get("runtime_state"))
+    session = save_runtime_session(session)
+
+    return {
+        "ok": True,
+        "scene_id": scene_id,
+        "action_type": action_type,
+        "beats": _safe_list(result.get("beats")),
+        "resolved": bool(result.get("resolved")),

Here is the Phase 7.3 — Scene Execution Layer file-by-file implementation spec for the coding LLM.

This phase turns your current control layer into a full gameplay loop:

ChoiceSet / ChoiceOption exist
GM and pacing/framing can shape options
now player choice must resolve into structured events
those events must flow through EventBus
CoherenceCore must absorb them
director/renderer then consume updated truth

The key rule for 7.3:

Option selection never mutates state directly.
It resolves into events, and existing reducers/system boundaries remain the authority path.

This preserves:

determinism
replay/snapshot safety
coherence authority
inspectability/debugging

Example implementation shape
def resolve_selected_option(self, option_id: str) -> dict:
    option = self.gameplay_control_controller.select_option(option_id)
    if option is None:
        return {"ok": False, "reason": "unknown_option", "option_id": option_id}

    result = self.action_resolver.resolve_choice(
        option=option,
        coherence_core=self.coherence_core,
        gm_state=self.gm_directive_state,
    )

    self._emit_action_resolution_events(result.to_dict())
    self._apply_coherence_updates_from_action_result(result.to_dict())

    return {
        "ok": True,
        "resolution": result.to_dict(),
        "scene_summary": self.coherence_core.get_scene_summary(),
    }
